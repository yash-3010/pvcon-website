## What is a PSMF?

The Pharmacovigilance System Master File (PSMF) is a detailed description of the pharmacovigilance system used by a Marketing Authorisation Holder. It is a living document that must be kept continuously updated and must be available to regulatory authorities upon request within seven days.

## Key Components of a PSMF

A compliant PSMF must include:

- Description of the pharmacovigilance system
- QPPV details and CV
- Organisational structure
- Sources of safety data
- Computerised systems and databases
- CAPA documentation
- Outsourced activities and contracts

## Common PSMF Deficiencies Found in Audits

Regulatory inspectors frequently cite the following issues during pharmacovigilance inspections:

1. Outdated QPPV information
2. Missing or incomplete SOPs referenced in the PSMF
3. Lack of evidence for ongoing training
4. Inadequate documentation of outsourced PV activities
